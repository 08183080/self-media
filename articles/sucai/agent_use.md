7 条与 AI Agent 协作的法则
01: Context, not control（给背景，而不是控制欲）
别去微调AI 的每一步，给它清晰的背景和最终目标，让它自己找路。它往往比你设想的更聪明。

02: Push until it breaks（把它逼到崩溃为止）
永远不要接受 AI 给的第一版答案。不断施压、要求变体，直到它犯错，你才能真正摸清它的边界在哪里。

03: Put in the reps（用次数喂出直觉）
别迷信什么“万能魔法提示词”，唯一的捷径就是疯狂练习。你折腾得越多，对 AI 的直觉就越准。

04: Try everything. Waste tokens.（大胆试，浪费 Token 也是种浪漫）
放开手脚去试那些看似离谱、毫无意义的想法。好奇心才是最宝贵的货币，毕竟 Token 会越来越便宜（当然也不一定。。。希望是）。

05: Build in public（公开你的构建过程）
别闭门造车，把进度分享出来，哪怕是那些乱糟糟的半成品。分享真实的过程，你能学得更快，也能找到同频的人。

06: Make it fun（保持乐趣）
如果和 AI 互动让你觉得枯燥沉重，那是你太严肃了。去享受它带来的那些意想不到的惊喜和幽默吧。

07: If you can talk, you can vibe code（只要会表达，你就能 Vibe Code）
别被黑乎乎的代码终端吓倒。现在最难的不再是写代码，而是你想清楚“要造什么”，以及如何让人为之共鸣。
